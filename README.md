# 🛡️ Professional Cybersecurity Analytics Dashboard

A professional Cybersecurity Analytics Dashboard built using **Python, Pandas, and Matplotlib** to monitor and visualize cybersecurity metrics through interactive charts and KPI cards.

## 🚀 Features

- 🛡️ Total Threats KPI
- 🚨 Critical Alerts KPI
- 🔐 Failed Login Attempts
- 🦠 Malware Detection Analysis
- 🌐 Network Traffic Monitoring
- 📈 Threat Trend Analysis
- 🥧 Threat Distribution
- 🌍 Country-wise Attack Analysis
- 💻 Device Risk Analysis
- 📉 Security Score Trend
- 🌙 Professional Dark Theme

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib

## ▶️ Run the Project

```bash
pip install -r requirements.txt
python cybersecurity_dashboard.py
```
